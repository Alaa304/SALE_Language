**ESAL Compiler** is a **simple IDE** for a custom educational programming language called `ESAL`.  
The project combines a **graphical code editor** and a **compiler** that detects errors, shows warnings, and prints a **Symbol Table** in real-time.

---

## üéØ Project Goal

- Design a simplified educational programming language (`ESAL`)  
- Build a **user-friendly IDE** for writing and running code  
- Display errors, warnings, and symbol table instantly  
- Teach **compiler basics** and **symbol table concepts**  
- Provide a mini IDE experience similar to real-world programming languages

---

## üìù Features

- Interactive code editor using **Tkinter**  
- ESAL language support:
  - Variable declaration (`##`)  
  - Arithmetic operations (`+ - * /`)  
  - Print statements (`@$`)  
  - Conditional statements (`@?`)  
  - Function definitions (`@@`)  
- Real-time **error and warning reporting**  
- Displays **Symbol Table**  
- Colorful, user-friendly GUI  
- Easily extendable for future features like loops and advanced functions
